Fresca JS Library

All of our projects are different, all of our JavaScript is different. Should we not instead build a new standard set of plugins into Blue Print? Essentially these will be locked files that can't be amended during a project.

If any amendments/improvements are required to a global plugin for a specific client, then they can be made using the jQuery extend functionality to override specific methods. Writing all our code as jQuery plugins will provide the encapsulation and reusability that is frequently missing from our code.
